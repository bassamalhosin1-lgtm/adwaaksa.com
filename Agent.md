## 🛠️ أوامر التشغيل
| الأمر | الوظيفة |
|-------|---------|
| `cd adwaaksa-site && python build.py` | بناء الموقع → مجلد `dist` |
| `automation\generate.bat "كلمة"` | توليد مقال + صورة تلقائياً |
| `automation\generate.bat --file keywords.txt` | توليد من ملف كلمات |
| `automation\generate.bat --auto-push` | توليد + رفع + نشر |
| `git add -A && git commit -m "msg" && git push` | نشر التغييرات |
