## 3️⃣ قواعد توليد المحتوى

> [!IMPORTANT]
> هذه القواعد ثابتة ومستخلصة من تجربة cabltexperts.com

1. **كل عنوان يحتوي على:** كابل/كيبل/كوابل أو التماس أو كشف أعطال
2. **كل وصف meta يذكر:** الكابلات + الرياض أو جدة + الخدمة
3. **المحتوى يركز 80%+** على: فحص، كشف، إصلاح، لحام كابلات
4. **كل مقال يبدأ بفقرة صدارة** (40-60 كلمة) — لاختطاف Featured Snippet
5. **كل مقال ينتهي بـ FAQ Schema** (سؤالين على الأقل)
6. **رقم 0558697397** يُذكر في بداية ونهاية كل مقال (CTA)
7. **تنويع المرادفات** (كابلات/كيابل/كوابل/كيبل) في كل فقرة
8. **المدن:** الرياض دائماً + جدة كسوق ثانوي

## 4️⃣ خطة SEO المتقدمة

### On-Page SEO
- H1 واحدة واضحة تحتوي الكلمة المستهدفة
- هيكل عناوين H2-H3 منطقي ومحشو بالكلمات
- ALT tags مُحسنة للصور
- Internal linking بين المقالات

### Technical SEO
- `sitemap.xml` تلقائي (يبنيه build.py)
- `robots.txt` تلقائي
- Canonical URLs
- Open Graph + Twitter Cards
- HTTPS (Cloudflare تلقائي)

### Local SEO
- LocalBusiness Schema كامل
- FAQ Schema في كل مقال
- AggregateRating Schema (129 تقييم من adwaaksa القديم!)
- Google Business Profile (يدوي)

## 5️⃣ خطة النشر

### CI/CD Pipeline
```
كتابة كود → git push → GitHub Actions → python build.py → wrangler pages deploy → Cloudflare Pages
```

### الخطوات:
1. إنشاء مستودع GitHub خاص: `bassamalhosin1-lgtm/adwaaksa.com`
2. إضافة GitHub Secrets: `CLOUDFLARE_API_TOKEN` + `CLOUDFLARE_ACCOUNT_ID`
3. إنشاء مشروع Cloudflare Pages: `adwaaksa-site`
4. ربط الدومين `adwaaksa.com` بالمشروع
5. أول `git push` → نشر تلقائي
